# warriors-fan

Chatbot to answer anything about the Golden State Warriors <br>
Note: this is not affiliated with the actual team

## How to use:

* Create .env file with single key named "API_KEY". Set this to be equal to any Claude AI API Key
* Import flask, Anthropic, and dotenv through pip
* Enter "python app.py" in terminal
* Open link http://127.0.0.1:5000/ on local machine
